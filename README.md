== 다육이 키우기

온실 내부의 온도,조도,습도 등의 환경을 파악하여
Fan, LED, 서보모터 등을 제어해서 
식물에게 최적의 환경을 만들어줄 수 있습니다.

자리를 너무 오래 비워서.. 식물을 키워도 곧 잘 시들어버렸기 때문에
이 프로젝트를 시작하게 되었다.

현재는 프로토타입 용도로 제작한 상태이다.

------------------------------------------------------------------------------

구성요소는 다음과 같다.
-------------------------------------------------------------------------------

Arduino Pro Mini 
+ LED
+ FAN (12V) + 릴레이모듈 + 9V 배터리
+ 서보모터
+ 토양습도센서
+ 온도센서(TMP36)
+ RTC
+ .. 그외 다수.
